# Setting up JS frontend projects
Configure frontend projects with unit test, integration test, bundling, TS and so on

## Requirements
* Nodejs 12.11+

## Setup
`npm i -g pnpm`
`pnpm i`

## Run
* `npm start`
* `npm test`
